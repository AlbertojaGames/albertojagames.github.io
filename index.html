FAQ

¿Unity me acepta esta web para “Publisher Website”?

Sí, porque es una web pública funcional con contenido y un método de contacto visible. Evita que el único objetivo sea redirigir a marketplaces.

¿Puedo enlazar mi Asset Store aquí?

Sí, pero mejor como enlace secundario cuando ya esté aprobado. Para pasar la revisión inicial, es más seguro no usar marketplaces como CTA principal.

¿Dónde pongo documentación y changelog?

Puedes subir un PDF o un README en el repo y enlazarlo. Si quieres, te preparo una página “docs.html” y una “changelog.html” para que quede aún más profesional.
