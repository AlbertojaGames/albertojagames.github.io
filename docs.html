Quick Start

5 minutos
  1. Importa el paquete en Unity (Assets → Import Package o Package Manager, según tu formato).
  2. Abre la escena de ejemplo (si incluye) o crea una escena vacía.
  3. Añade el objeto “DungeonGenerator” (prefab) a la escena.
  4. Configura la Seed y pulsa Generate.
  5. Guarda la escena o exporta el layout si tu herramienta lo permite.

Requisitos

  • Unity: (pon aquí tu versión mínima recomendada)
  • Pipeline: Built-in / URP / HDRP (indica compatibilidad real)
  • Plataforma: Windows/Mac (y otras si aplica)

Conceptos

  • Seed: misma semilla = mismo layout.
  • Rooms/Corridors: reglas de colocación y conexión.
  • Multi-Floor: cada planta genera su layout y se conecta con escaleras.

generator.Floors = 3;
generator.Generate();
